# Alfred AI Assitant

![Alfred](https://upload.wikimedia.org/wikipedia/en/5/5c/Alfred_Pennyworth_%28Alex_Ross%29.jpg?utm_source=en.wikipedia.org&utm_campaign=index&utm_content=original)

It is a voice assistant with built-in local AI that uses Ollama. You can modify variables in the .env file.

## Installation

1. Clone the repository
```bash
git clone https://github.com/RykerWilder/jarvis
```

2. Change directory
```bash
cd alfred
```

3. Run setup script
```bash
bash setup.sh
```

Now you are ready to use Alfred, run `python3 main.py`.

## Tools

Alfred already has some tools inside including:
(* if you are connected to a network)

- Check internet connection
- Speedtest *
- Play music on youtube *
- Get current time
- Make a search on Google *
